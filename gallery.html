¿Qué es Bedsense?

Bedsense es un sistema de gestión en tiempo real de camas hospitalarias que permite a los administradores de hospitales monitorear la disponibilidad de camas por área. El software proporciona una vista clara de cuántas camas están ocupadas y cuántas están libres, mejorando la eficiencia operativa y la toma de decisiones en el manejo de recursos hospitalarios.

Características

  • Monitoreo en tiempo real por área hospitalaria
  • Seguimiento eficiente de camas disponibles y ocupadas
  • Almacenamiento robusto con Google Cloud SQL y MySQL

Tecnologías utilizadas

  • Python para la lógica del sistema
  • Google SQL + MySQL para la base de datos

¿A quién va dirigido?

Este sistema está diseñado para administradores de hospitales, personal médico y responsables de la gestión de recursos hospitalarios, permitiendo un manejo más eficiente y transparente de la disponibilidad de camas.
